# Changelog

All notable changes to `rmscms/accounting` will be documented in this file.

## [1.0.0] - 2025-01-08

### Added
- Initial release
- Complete Double-Entry Accounting System
- Ledger-First Design with Immutable Ledger
- Chart of Accounts (General/Subsidiary/Analytical)
- Multi-Currency Support with FX Rate Management
- Accounts Receivable (Customer Invoices & Payments)
- Accounts Payable (Supplier Invoices & Payments)
- Purchase Order Management
- Treasury Management (Banks, Cash Boxes, POS, Cheques, Wallets)
- Expense Management System
- VAT System with Tax Rates
- COGS (Cost of Goods Sold) Tracking
- Payment Reconciliation System
- Settlement System
- Fiscal Year Management
- 50+ Financial Reports
- Service API for external systems
- Admin API for management panel
- Artisan Commands for automation
- Real-time Balance Calculation from Ledger
- Event-Driven Architecture
- Multi-Store Support

### Features
- 30 Database Tables + 2 VIEWs

- 20+ Admin Controllers
- 5+ Service API Controllers
